DataTables

Tambahan kontrol interaktif pada tabel HTML yang mudah digunakan.

Kunjungi DataTables

Default

Method ini terjadi ketika parameter config kosong atau tidak diisi. Method default sudah dirancang di mana kolom pertama dan terakhir pada tabel dinonaktifkan aksi mengurutkannya.

Jadi biasanya method ini digunakan ketika kolom pertama pada tabel berisi ceklis, dan kolom terakhir pada tabel berisi tombol-tombol aksi pada data yang disajikan.

See the Pen Spandiv: DataTables 1 by Aji Fatur (@ajifatur) on CodePen.


Order All

Method ini terjadi ketika parameter config diisi pada bagian properti orderAll dengan nilai true. Jadi kita bisa mengurutkan data pada tabel dari semua kolom yang ada.

See the Pen Spandiv: DataTables 2 by Aji Fatur (@ajifatur) on CodePen.


Page Length

Method ini terjadi ketika parameter config diisi pada bagian properti pageLength dengan nilai 25, 100, atau -1. Method ini berfungsi menampilkan jumlah data dalam tabel. Nilai defaultnya adalah 10.

See the Pen Spandiv: DataTables 3 by Aji Fatur (@ajifatur) on CodePen.
